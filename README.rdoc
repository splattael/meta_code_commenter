= Meta code commenter (mcc)

Comments your meta code.

== Usage

  require 'mcc'

  params = ARGV.inject({}) do |hash, arg|
    key, value = arg.split(/=/)
    hash[key] = value
    hash
  end

  puts MetaCodeCommenter.comment(STDIN.readlines)

=== Command line

  $ mcc key1=value key2=value < my_meta_code

== Installation (not yet)
